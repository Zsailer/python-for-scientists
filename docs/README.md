We still need to improve the plotting module.

```
.. raw:: html

  <div class="terminal-window">
    > pip install numpy
  </div>

```

To resize the terminal
```


.. raw:: html

  <div class="terminal-window" style="height:100px; width:300px;">
    > pip install numpy
  </div>

```
